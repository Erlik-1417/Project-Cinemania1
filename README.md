# Project Cinemania

Bu proje Vite kullanılarak geliştirilmektedir.

## Kurulum Adımları
1. Projeyi klonlayın: `git clone <repo-linki>`
2. Bağımlılıkları yükleyin: `npm install`
3. `.env.example` dosyasının adını `.env` yapın ve TMDB API anahtarınızı ekleyin.
4. Projeyi lokalde çalıştırın: `npm run dev`

## Git Kuralları
- Doğrudan `main` branch'ine push yapmak yasaktır.
- Her görev için `feature/gorev-adi` şeklinde yeni bir branch açın.
- Göreviniz bittiğinde `main` branch'ine Pull Request (PR) açın ve Team Lead'e haber verin.
